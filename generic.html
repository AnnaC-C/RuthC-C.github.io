Advanced Game Technologies

I created a physics simulation scene utilising and adding on to the Physics Engine that Newcastle University provided - NCLTech. I implemented a variety of techniques to create this scene, including networking, gravity, constraints to make a complex object, scoring between each player with an external scoreboard, a second planet in orbit and an atmosphere that slows every object inside it.

  • The main purple planet has gravity implemented on it.

  • A scoring system on the target that is attached to the planet.

  • The atmosphere on the planet slows the linear velocity of every object inside it.

  • Projectile being shot from the player.

  • A peer being passed from the server.

  • All the objects are in rest state before they are hit by another physics object.

  • Collision volumes of each object.

  • Collision normals of each object.
