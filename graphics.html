Advanced Graphics for Gaming

Using advanced proficiency in the C++ language, I created a scene using multiple Graphics techniques. I achieved these using the Graphics Framework that Newcastle University provided - NCLGL, based on OpenGL.

  • Heightmap for the ground using simplexnoise.

  • Lighting with shadow mapping.

  • Dynamic directional light.

  • Skybox

  • Particle Effects (explosion).

  • Particle Effects (rain falling)
